About the project

PA-CoT is a multi-stage prompting method for health and nutrition assistants that operate on structured user profiles — demographics, goals, dietary patterns, and nutritional statistics. It treats profile interpretation as a dedicated stage before generation, and runs an independent safety-verification stage afterward.

The method is introduced alongside QPA (Question–Profile–Answer), an evaluation format that pairs free-form questions with structured profiles and reference answers, reflecting how real products accumulate and pass user context at inference time.

Headline results

On 200 QPA samples (G-Eval, 1–5 scale), PA-CoT leads on both criteria it was designed for, with non-overlapping 95% confidence intervals over the nearest competitor.
